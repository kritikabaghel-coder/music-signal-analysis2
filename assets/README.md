# 📸 Screenshots Folder

This folder contains screenshot images used in the README documentation.

## Expected Image Files

Please save the following screenshot images to this folder:

1. **home.png** - Home Interface showing the app title, subtitle, and Features card
2. **upload.png** - Upload & Analysis tab with file uploader section
3. **results.png** - Analysis Results showing metrics (Duration, BPM, Genre, Confidence)
4. **visualizations.png** - Visualizations tab with waveform and spectrogram plots
5. **advanced.png** - Advanced Signal Analysis tab with expandable sections

## Image Requirements

- Format: PNG
- Width: 800px (for optimal display in README)
- Quality: High resolution (screenshots at 1x or 2x scale)
- Include full UI context (header, tabs, content card)

## Adding Images

1. Take screenshots of the Streamlit app running at `http://localhost:8501`
2. Save each screenshot as PNG format
3. Place in this `assets/` folder with the names listed above
4. Commit to git: `git add assets/*.png && git commit -m "Add screenshots"`
