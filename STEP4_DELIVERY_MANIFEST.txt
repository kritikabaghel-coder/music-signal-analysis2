CORE SIGNAL PROCESSING MODULES (Production Code - 700 lines):
──────────────────────────────────────────────────────────────

✅ beat_detector.py (300 lines)
   Class: BeatDetector
   
   Core Methods:
     • load_audio(file_path) → y, sr
       - Load WAV files with librosa
     
     • compute_onset_strength(y, sr) → np.ndarray
       - Frequency domain onset detection
       - STFT-based envelope
     
     • detect_beats(y, sr) → beat_frames, tempo
       - Core beat tracking (librosa.beat.beat_track)
       - Returns frames and BPM
     
     • convert_frames_to_time(frames, sr) → np.ndarray
       - Frame indices → time in seconds
     
     • extract_beat_info(file_path, genre) → dict
       - Single file complete extraction
       - Returns all beat information
     
     • extract_beats_batch(file_list, genres) → list[dict]
       - Batch process multiple files
       - Verbose progress reporting
     
     • compute_beat_density(beat_times, duration) → float
       - Beats per second metric
     
     • estimate_beat_regularity(beat_times) → float
       - Coefficient of variation
       - Measures consistency of beat intervals
     
     • get_statistics(beat_results) → dict
       - Aggregate statistics from results
       - Per-genre breakdowns
     
     • print_statistics(stats) → None
       - Formatted console output
   
   Status: ✅ Production-ready


✅ beat_pipeline.py (400 lines)
   Class: BeatExtractionPipeline
   
   Purpose: Orchestrate and analyze beat detection across dataset
   
   Key Methods:
     • process_dataset(df_metadata, output_path) → DataFrame
       - Load all files
       - Process each file
       - Save results to CSV
     
     • compute_statistics() → dict
       - Calculate statistics from results
     
     • print_results(n_samples) → None
       - Print sample results and statistics
     
     • get_tempo_by_genre() → DataFrame
       - Average BPM per genre
       - With std, min, max, count
     
     • get_high_tempo_tracks(threshold, top_n) → DataFrame
       - Get fastest tracks
     
     • get_low_tempo_tracks(threshold, top_n) → DataFrame
       - Get slowest tracks
     
     • compare_genres_by_tempo() → None
       - Formatted genre comparison
     
     • plot_tempo_distribution() → (fig, ax)
       - Histogram + boxplot visualization
     
     • plot_beat_waveform(file_idx) → (fig, ax)
       - Waveform with beat overlays
   
   Status: ✅ Production-ready
