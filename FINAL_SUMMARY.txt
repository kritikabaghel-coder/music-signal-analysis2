TECHNICAL IMPLEMENTATION DETAILS
═════════════════════════════════════════════════════════════════════════════════

SIGNAL PROCESSING FLOW:
┌─────────────────────────────────────────────────────────────────┐
│ 1. Audio Loading                                                │
│    └─ librosa.load(file_path, sr=None, mono=True)             │
│       - Preserves original sample rate                          │
│       - Converts to mono                                        │
│       - Handles multiple audio formats                          │
└─────────────────────────────────────────────────────────────────┘
         ↓
┌─────────────────────────────────────────────────────────────────┐
│ 2. FFT (freq domain) → STFT (time-freq) → Mel scale → Log      │
│                                                                  │
│    FFT: y[t] ──→ FFT ──→ Y[f]                                 │
│    STFT: y[t] ──→ Window+FFT ──→ Spectrogram                  │
│    Mel: Spectrogram ──→ Mel-scale ──→ 128 bins                │
│    Log: Linear ──→ dB scale (log)                              │
└─────────────────────────────────────────────────────────────────┘
         ↓
┌─────────────────────────────────────────────────────────────────┐
│ 3. Feature Extraction (9 Methods)                               │
│    ├─ Magnitude spectrum analysis                               │
│    ├─ Spectral centroid (center of mass)                        │
│    ├─ Spectral bandwidth (spread)                               │
│    ├─ Zero-crossing rate                                        │
│    ├─ Chroma features (12 pitch classes)                        │
│    ├─ MFCC (Discrete Cosine Transform on Mel)                 │
│    ├─ Spectral rolloff (energy threshold)                       │
│    └─ Optional: Contrast, tempogram, CQT                        │
└─────────────────────────────────────────────────────────────────┘
         ↓
┌─────────────────────────────────────────────────────────────────┐
│ 4. Feature Vector Integration                                   │
│    └─ Combine all 80+ features into single vector              │
│       Dimensions: (1, 82)                                       │
└─────────────────────────────────────────────────────────────────┘
         ↓
┌─────────────────────────────────────────────────────────────────┐
│ 5. Normalization (StandardScaler)                               │
│    └─ x_normalized = (x - mean) / std                          │
│       Zero mean, unit variance per feature                      │
└─────────────────────────────────────────────────────────────────┘
         ↓
┌─────────────────────────────────────────────────────────────────┐
│ 6. Feature Matrix                                               │
│    └─ Stack all feature vectors (1000 files)                   │
│       Dimensions: (1000, 82)                                    │
│       Export: CSV format                                        │
└─────────────────────────────────────────────────────────────────┘
