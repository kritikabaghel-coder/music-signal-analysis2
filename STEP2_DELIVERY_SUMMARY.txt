═════════════════════════════════════════════════════════════════════════════
TECHNICAL IMPLEMENTATION
═════════════════════════════════════════════════════════════════════════════

FFT (Fast Fourier Transform):
  ✓ Decompose signal into frequency components
  ✓ Extract magnitude spectrum
  ✓ Identify dominant frequencies
  ✓ Compute statistics: mean, std, max

STFT (Short-Time Fourier Transform):
  ✓ Apply FFT to overlapping windows
  ✓ Create time-frequency representation
  ✓ Window size: 2048 samples
  ✓ Hop length: 512 samples

Mel Spectrogram:
  ✓ Apply Mel-filterbank (128 bins)
  ✓ Follows human frequency perception
  ✓ Convert to dB scale (logarithmic)
  ✓ Extract statistics

MFCC (Mel-Frequency Cepstral Coefficients):
  ✓ Compute Mel spectrogram
  ✓ Apply DCT (Discrete Cosine Transform)
  ✓ Extract 13 coefficients
  ✓ Time-average over duration

Spectral Centroid:
  ✓ Center of mass of frequency distribution
  ✓ Indicates "brightness" of sound
  ✓ Higher = brighter, Lower = darker

Spectral Bandwidth:
  ✓ Spread of spectrum around centroid
  ✓ High = broad/noisy spectrum
  ✓ Low = narrow/tonal spectrum

Zero-Crossing Rate:
  ✓ Count signal sign changes
  ✓ High ZCR = noisy/unvoiced
  ✓ Low ZCR = tonal/voiced

Chroma Features:
  ✓ Extract pitch class content
  ✓ 12 pitch classes (C through B)
  ✓ Useful for music harmony analysis

Normalization:
  ✓ StandardScaler: (x - mean) / std
  ✓ Zero mean, unit variance
  ✓ Applied per feature
