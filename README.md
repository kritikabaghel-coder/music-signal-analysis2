# 🎵 Music Signal Analysis System

A web-based application for analyzing audio signals using spectral methods and beat detection.
It extracts meaningful audio features and visualizes them through an interactive dashboard.

---

## 🚀 Live Demo

👉 [Try the App](https://music-signal-analysis2-3epbxlxazrdylh4tq65b93.streamlit.app)

---

## 📌 Overview

This project focuses on **music signal processing** and visualization.
It allows users to upload an audio file and analyze its characteristics using techniques like **spectral analysis, waveform visualization, and tempo detection**.

---

## ✨ Features

* 🎵 Upload audio files (MP3, WAV, OGG, FLAC)
* 📊 Waveform visualization (time-domain analysis)
* 🌈 Spectrogram visualization (frequency-domain analysis)
* ⏱️ Tempo (BPM) detection using beat tracking
* 📈 Advanced signal features:

  * Spectral Centroid
  * Spectral Bandwidth
  * RMS Energy
  * Frequency Distribution (FFT)
  * Chromagram
* 🎨 Clean and interactive Streamlit UI
